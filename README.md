## Dot files for:
- Tmux
- Zsh
- vim
- Pow
- Git
- Rspec
- Postgres client
- Ruby gem installation (disable docs)


### Bin files
- ```vol``` opens the newest file in a directory in your $EDITOR. Usage: ```vol dotfiles/bin```

### Installation

```
git clone https://github.com/bjensen/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```
